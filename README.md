# Teslo API

## Steps to start the server

1. Clone the repo

2. Install dependencies

```
  yarn install
```

3. Copy `.env.template` file into `.env`

4. Update env variables

5. Start the DB

```
  docker-compose up -d
```

6. Start the Server

```
  yarn start:dev
```

7. Excecute SEED

```
http://localhost:3000/api/seed
```
